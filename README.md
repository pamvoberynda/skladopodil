Проєкт для автоматичного поділу українських слів на склади на основі фонетичних правил О. Тоцької (теорія сонорності / хвилі звучності). Програма також має графічний інтерфейс користувача (GUI) на базі Gradio.

🛠️ Структура проєкту
.
├── syllabizer.py      # Основна логіка складоподілу
├── visual_image.py    # Gradio-інтерфейс для зручного використання
└── rules.md           # Правила складоподілу за О. Тоцькою
🔬 Як це працює (Індекси звучності)

Кожна літера слова переводиться в цифровий індекс від 0 до 4:

    0 — Глухі приголосні (ц, с, к, ф, п...)
    1 — Дзвінкі приголосні (д, г, ґ, з...)
    2 — Сонорні приголосні (й, н, м, л, р, в)
    3 — Подовжені приголосні / м'який знак (ь, нн, лл...)
    4 — Голосні (а, я, о, е, є...)


Кроки алгоритму
Оцифирування (digitilize_word) — слово перетворюється на рядок індексів звучності.
Первинний поділ (mk_raw_syllables) — межі складів проводяться там, де індекс зменшується (спадна звучність).
Перегляд і корекція (review_raw_syllables) — застосовуються правила О. Тоцької: обробка сонорних, перевірка наявності голосної в кожному складі, злиття неповних складів.
Співвіднесення з оригіналом (correlate_review_word) — цифровий поділ накладається на оригінальний текст.


🚀 Запуск та використання
1. Встановлення залежностей
Для роботи графічного інтерфейсу потрібна бібліотека gradio:
    Bash
    pip install gradio

2. Запуск у консолі (демо-вірш)

Щоб побачити тестовий приклад роботи алгоритму на вірші, запустіть:
    Bash
    python syllabizer.py

3. Запуск веб-інтерфейсу

Щоб відкрити зручну сторінку в браузері:
    Bash
    python visual_image.py

Після цього перейдіть за локальним посиланням, яке з'явиться в консолі (зазвичай http://127.0.0.1:7860).

📝 Приклад роботи

Вхідний текст: > Як прекрасна дівчина посміхнулась мені.

Результат складоподілу:
    як пре-кра-сна ді-вчи-на по-смі-хну-лась ме-ні.

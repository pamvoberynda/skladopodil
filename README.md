Проєкт для автоматичного поділу українських слів на склади на основі фонетичних правил О. Тоцької. Програма також має графічний інтерфейс користувача (GUI) на базі Gradio.

###  Структура проєкту
```
.
├── syllabizer.py      # Основна логіка складоподілу
├── visual_image.py    # Gradio-інтерфейс для зручного використання
└── rules.md           # Правила складоподілу за О. Тоцькою
```

### Індекси звучності

Кожна літера слова переводиться в цифровий індекс від 0 до 4:
```
    0 — Глухі приголосні (ц, с, к, ф, п...)
    1 — Дзвінкі приголосні (д, г, ґ, з...)
    2 — Сонорні приголосні (й, н, м, л, р, в)
    3 — Подовжені приголосні / м'який знак (ь, нн, лл...)
    4 — Голосні (а, я, о, е, є...)
    5 - Будь-який інший небуквенний символ  
```

### Кроки алгоритму
1. **Оцифирування** (digitilize_word) — слово перетворюється на рядок індексів звучності.
2. **Первинний поділ** (mk_raw_syllables) — межі складів проводяться там, де індекс зменшується (спадна звучність).
3. **Перегляд і корекція** (review_raw_syllables) — застосовуються правила О. Тоцької: обробка сонорних, перевірка наявності голосної в кожному складі, злиття неповних складів.
4. **Співвіднесення з оригіналом** (correlate_review_word) — цифровий поділ накладається на оригінальний текст.


### Запуск та використання
1. Встановлення залежностей
Для роботи графічного інтерфейсу потрібна бібліотека gradio:
```Bash
    pip install gradio
```

2. Запуск у консолі (демо-вірш)

Щоб побачити тестовий приклад роботи алгоритму на вірші, запустіть:
```Bash
    python syllabizer.py
```

3. Запуск веб-інтерфейсу

Щоб відкрити сторінку в браузері:
```Bash
    python visual_image.py
```

Після цього перейдіть за локальним посиланням, яке з'явиться в консолі (зазвичай http://127.0.0.1:7860).

### Приклад роботи

Вхідний текст: 
```
Ще зима не минула, я побачив тоді,
Як прекрасна дівчина посміхнулась мені.
```

Результат складоподілу:
```
Ще зи-ма не ми-ну-ла, я по-ба-чив то-ді,
Як пре-кра-сна дів-чи-на по-смі-хну-лась ме-ні.
```
